Endpoints:

For User:
1. http://localhost:3000/api/signup [name, email, password]
2. http://localhost:3000/api/login
3. http://localhost:3000/api/logout 

For books:
1. http://localhost:3000/api/books/create  --> To add books [title, author, genre]
2. http://localhost:3000/api/books/get --> To get all available books 

For posts:
1. http://localhost:3000/api/post/create --> To create posts 
2. http://localhost:3000/api/post/get --> To get all the available posts
